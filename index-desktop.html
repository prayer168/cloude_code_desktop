詳細步驟

完整開發流程

點擊展開各步驟的說明與操作細節

1
下載與安裝 Desktop App
官方安裝檔 · macOS / Windows · 首次登入
▼
⚠️ 系統需求
  • macOS：通用版（Intel + Apple Silicon）
  • Windows：x64 或 ARM64（不支援 Linux，Linux 請用 CLI 版）
  • 帳號：Claude Pro($20/月)以上，或 Max / Team / Enterprise

安裝步驟：

  1. 前往 https://claude.com/download 下載對應平台的安裝檔
  2. 執行安裝程式，從 Applications（macOS）或開始選單（Windows）啟動
  3. 登入 Claude 帳號
  4. 點擊頂部中央的 Code 分頁
  5. Windows 使用者：若 Code 分頁無法開啟，需先安裝 Git for Windows，安裝後重啟 App
✅ Desktop App 內建 Claude Code

不需要另外安裝 Node.js 或 CLI。若你也想從終端機使用 claude 指令，才需要另行安裝 CLI 版。

2
開啟第一個 Session
選擇環境 · 選擇資料夾 · 授權模式
▼

進入 Code 分頁後，在輸入框上方依序設定四個項目，再輸入任務：

設定項目建議選擇說明
執行環境Local直接使用本機檔案，教材開發選此項
專案資料夾你的教材目錄點擊 Select folder 選擇
模型Sonnet（預設）Session 中可隨時切換
授權模式Ask（預設）每次修改都要你確認，初學者推薦

選擇授權模式（Permission Mode）— 可依作業需求隨時切換：

Ask（預設）
Auto Accept
Plan
🔍 Ask 模式（建議初學者使用）

Claude 每次修改檔案都需要你確認。會顯示 Diff 視圖，你可以逐行接受或拒絕。如果拒絕，Claude 會詢問你希望如何調整。你的檔案在你按下 Accept 之前不會被修改。

⚡ Auto Accept Edits 模式

Claude 自動接受所有檔案修改，無需逐一確認。適合快速迭代、微小調整，或你已充分了解 Claude 在做什麼的情況下使用。仍可隨時打斷 Claude。

🗺️ Plan 模式

Claude 只規劃、不修改任何檔案。適合複雜任務開始前，先審查 Claude 的執行計畫，確認方向正確後再切換到 Ask 或 Auto 模式執行。

3
建立 CLAUDE.md 專案記憶
最關鍵步驟 · GUI 操作 · 範本可直接使用
▼

在 Code 分頁的輸入框中輸入以下內容，讓 Claude 幫你建立 CLAUDE.md：

Code 分頁輸入框 
請幫我建立 CLAUDE.md，說明以下專案規範：
1. 本專案為國小自然科互動教材，基於 108 課綱
2. 使用純 HTML5 + CSS3 + Vanilla JavaScript
3. 輸出為單一 .html 檔案，不依賴任何外部資源
4. 所有文字使用繁體中文，程式碼註解也用中文
5. 適合 8-12 歲學生，按鈕至少 44px，支援觸控操作

Claude 生成後，在 Diff 視圖中確認內容後按 Accept 建立檔案。

💡 為什麼這是最關鍵步驟？

CLAUDE.md 是 Claude Code 的「永久記憶」。每次開啟新 Session 時都會自動讀取，讓 Claude 始終了解你的技術規範與目標對象，不需要每次重新說明。

4
生成互動教材
提示語範本 · 視覺 Diff 審查 · 即時預覽
▼

在 Code 分頁輸入教材需求，Claude 會自動生成 HTML 檔案並等候你的 Diff 審查：

基本範本 
請幫我設計互動教材 index.html，主題是「植物的光合作用」，
適合國小四年級學生。

需求：
1. 頂部：動畫展示光合作用過程（陽光 + 水 + CO₂ → 氧氣 + 葡萄糖）
2. 中間：可點擊的葉片示意圖，點擊不同部位顯示說明
3. 下方：5 題互動測驗，選擇後立即顯示對錯與解說
4. 全部繁體中文，溫暖的黃綠色系，卡通風格
5. 單一 HTML 檔案，不需要外部資源
🔍 生成後如何使用 Diff 視圖
  • 生成完成後，點擊輸入框旁的 +N -N 指示器開啟 Diff 視圖
  • 綠色行 = 新增內容，紅色行 = 刪除內容
  • 逐檔案審查，確認後按 Accept（或 Reject 並告訴 Claude 如何改）
  • 點擊 Review code 讓 Claude 自我審查，添加行內建議
5
迭代優化 · 並行 Session
持續修改 · 多單元並行開發
▼

常見迭代對話（直接在輸入框輸入，不需要指令）：

Code 分頁對話範例 
動畫速度太快，請調慢到原來的一半
測驗解說太簡短，每題加入 2-3 句詳細說明
在頁面頂部加入「回首頁」按鈕，連結 ../index.html
確認所有 CSS 動畫加入 -webkit- 前綴，相容 Safari

使用並行 Session 同時開發多個教材單元：

  1. 點擊側邊欄上方的 + 新增 Session
  2. 為新 Session 選擇不同的專案資料夾（如 units/solar-system）
  3. 各 Session 的 Chat 記錄、修改內容完全獨立不相干
  4. 在側邊欄快速切換，監控所有 Session 的進度
💡 打斷與修正 Claude

若 Claude 正在往錯誤的方向執行，可以隨時點擊停止按鈕，或直接輸入修正意見並按 Enter。Claude 會立刻停止當前操作並根據你的指示調整。

請幫我設定 SSH 金鑰連結到 GitHub，
我的 GitHub Email 是 your@email.com
首次推送（請 Claude 操作） 
請幫我：
1. 初始化 Git 倉庫並建立 .gitignore
2. 建立首次 commit
3. 連結到 github.com:USERNAME/REPO.git
4. 推送到 main 分支

  • 倉庫 → Settings → Pages
  • Source：Deploy from a branch，Branch：main，目錄：/ (root)
  • Save → 等 1-3 分鐘
  • 網址：https://USERNAME.github.io/REPO-NAME/
